## 关于您在 GitHub 上的个人账户

要开始使用 GitHub，您需要创建一个免费的个人账户并验证您的电子邮件地址。

在创建账户时，您还可以使用支持的社交登录提供商进行身份验证（目前仅支持 Google）。

每个使用 GitHub 的人都会登录到一个用户账户。您的用户账户是您在 GitHub 上的身份，拥有用户名和档案。例如，参见 @octocat 的档案。

之后，您可以探索 GitHub 提供的不同类型的账户，并决定是否需要付费计划。更多信息，请参见 GitHub 账户类型和 GitHub 计划。

请注意，本文中的步骤不适用于企业托管用户。如果您的 GitHub 账户是由您的公司为您创建的，则可以跳过本文并继续阅读《Hello World》。

## 注册新个人账户

1. 导航到 https://github.com/。
2. 点击"Sign up"（注册）。
3. 或者，点击"Continue with Google"（使用 Google 继续）以使用社交登录进行注册。
4. 按照提示创建您的个人账户。

在注册过程中，系统会要求您验证电子邮件地址。如果没有经过验证的电子邮件地址，您将无法完成某些基本的 GitHub 任务，例如创建仓库。

一些企业会为其用户创建托管用户账户。您不能使用已为托管用户账户验证的电子邮件地址注册个人账户。

如果您在验证电子邮件地址时遇到问题，可以采取一些故障排除步骤。更多信息，请参见验证您的电子邮件地址。

## 后续步骤

- 既然您已经创建了个人账户，我们将开始探索 GitHub 的基础知识。在下一个教程《Hello World》中，您将了解仓库以及如何创建仓库，并将接触到分支、提交和拉取请求等概念。
- 我们强烈建议您为账户配置双因素身份验证（2FA）。2FA 是额外的安全层，可以帮助保护您的账户安全。更多信息，请参见配置双因素身份验证。
